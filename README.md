📌 Repository Description
This project demonstrates the implementation of a Decision Tree Classifier for handwritten digit recognition using the MNIST dataset.
The workflow includes data preprocessing, normalization, reshaping of image data into feature vectors, model training, and performance evaluation. The model is trained using entropy as the splitting criterion and evaluated using accuracy, confusion matrix, and classification report.

🚀 Key Features
Load and explore the MNIST dataset
Preprocess image data (normalization & flattening)
Train a Decision Tree model using Scikit-learn
Evaluate model performance with:
Accuracy score
Confusion matrix visualization
Classification report (precision, recall, F1-score)
Visualize predictions and sample images

🛠️ Technologies Used
Python
NumPy
Matplotlib & Seaborn
Scikit-learn
TensorFlow/Keras (for dataset loading)

🎯 Goal
The goal of this project is to provide a simple and clear example of applying machine learning techniques to image classification problems.
